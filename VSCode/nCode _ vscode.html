vscode

1) Installation

Télécharger vscode à partir du site officiel

Ouvrez un terminal Atl Ctrl T

Rendez-vous dans le répertoire des téléchargements cd Téléchargements

Lancez l'installation de vscode sudo apt install ./FICHIER.deb

Raccourcis clavier

Ctrl b: Affiche ou masque le panneau de gauche

Ctrl j: Affiche ou masque le panneau du bas (Terminal)

Ctrl k Ctrl o: Ouvrir un dossier

Ctrl d: sélectionne l'élément suivant

Alt flèche: déplace une ligne ou la sélection

Alt click: positionne un nouveau curseur (ou select midle click)

Ctrl roulette: Zoom dans l'editeur

Ctrl + ou Ctrl -: Zoom l'interface

Ctrl K Z: mode zen
